# Somatic Project

## File mapping

- `somatic-manifesto.html` is the source file for `index.html` (home page at `/`)
- `somatic-product.html` is the source file for `life/index.html` (product page at `/life`)

When the user references `somatic-manifesto.html`, apply changes to `index.html`.
When the user references `somatic-product.html`, apply changes to `life/index.html`.
